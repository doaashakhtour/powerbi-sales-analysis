 📊 Power BI Sales Analysis Dashboard

## 📌 Project Overview

This project is a Power BI dashboard created to analyze sales performance, customer behavior, product performance, and profitability using a real-world sales dataset from Kaggle.

The goal is to transform raw sales data into meaningful business insights through data cleaning, modeling, DAX calculations, and interactive visualizations.

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing products
- Understand customer purchasing behavior
- Compare sales across regions and time
- Measure profitability using key business metrics
- Build an interactive dashboard for decision-making

---
